📋 Setup Guide

Connect Google Sheets in 5 minutes: 1. Open Google Sheets → Extensions → Apps Script
2. Paste the full contents of apps-script.gs
3. Run bootstrapSheets() once to create all sheets + seed data
4. Click Deploy → New Deployment → Web App
5. Execute as: Me | Access: Anyone (even anonymous)
6. Copy the Web App URL and paste above
7. Hit Save & Connect — done!
